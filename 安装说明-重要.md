# ⚠️ Node.js 安装说明

## 当前状态
系统检测到您的Mac上**未安装 Node.js**，这是运行本项目所必需的。

## 🚀 快速安装方法（推荐）

### 方法1：官网下载安装（最简单，推荐）

1. **打开浏览器**，访问：https://nodejs.org/zh-cn/
2. **下载 LTS 版本**（长期支持版，更稳定）
3. **双击下载的 .pkg 文件**
4. **按照安装向导完成安装**（通常只需要点击"继续"和"安装"）
5. **安装完成后，重新打开终端（Terminal）**
6. **验证安装**：
   ```bash
   node --version
   npm --version
   ```
   如果显示版本号，说明安装成功！

7. **然后运行项目**：
   ```bash
   cd "/Users/zade/Desktop/JW 2"
   ./start.sh
   ```

### 方法2：使用 Homebrew（如果已安装）

如果您已经安装了 Homebrew，可以在终端运行：

```bash
brew install node
```

## 📋 安装完成后的步骤

安装 Node.js 后，在终端运行以下命令：

```bash
# 1. 进入项目目录
cd "/Users/zade/Desktop/JW 2"

# 2. 运行启动脚本（会自动安装依赖和初始化数据库）
./start.sh
```

或者手动执行：

```bash
cd "/Users/zade/Desktop/JW 2"
npm install          # 安装项目依赖
node init-db.js     # 初始化数据库
npm start           # 启动服务器
```

## ✅ 验证安装

安装完成后，在终端运行：

```bash
node --version
npm --version
```

**应该显示类似：**
```
v18.17.0
9.6.7
```

## ❓ 常见问题

### Q: 安装后还是提示找不到 node？
A: 请**重新打开终端**，或者运行：
```bash
source ~/.zshrc
# 或
source ~/.bash_profile
```

### Q: 安装很慢怎么办？
A: Node.js 安装包大约 50MB，如果下载慢，可以：
- 使用手机热点
- 稍后重试
- 使用国内镜像（如果使用 Homebrew）

### Q: 需要管理员权限吗？
A: 安装 Node.js 时需要输入 Mac 登录密码（不是 root 密码）

## 💡 提示

- Node.js 安装是一次性的，安装后可以一直使用
- 建议安装 LTS（长期支持）版本，更稳定
- 安装过程通常只需要 2-3 分钟
