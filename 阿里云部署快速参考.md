## 📋 部署流程总览

```
1. 购买阿里云服务器 (5分钟)
   ↓
2. 连接服务器 (2分钟)
   ↓
3. 运行一键部署脚本 (3分钟)
   ↓
4. 配置安全组 (2分钟)
   ↓
5. 完成！
```

---

## 步骤1：购买阿里云服务器（5分钟）

1. **访问阿里云**
   - 打开：https://www.aliyun.com
   - 注册并完成实名认证

2. **购买服务器**
   - 进入 **云服务器ECS** → **立即购买**
   - 选择配置：
     - 地域：选择最近的
     - 实例：**1核2GB**（够用）
     - 镜像：**Ubuntu 22.04 64位**
     - 系统盘：**40GB**
     - 公网IP：**分配公网IPv4地址**
     - 带宽：**1Mbps**
     - 设置密码：**记住这个密码！**
   - 点击 **确认订单** 并支付

3. **获取服务器信息**
   - 进入 **云服务器ECS控制台** → **实例**
   - 记录：
     - **公网IP**：`47.xxx.xxx.xxx`
     - **用户名**：`root`
     - **密码**：你刚才设置的密码

---

## 步骤2：连接服务器（2分钟）

### Windows 用户

1. **下载 PuTTY**
   - 访问：https://www.putty.org
   - 下载 `putty.exe`

2. **连接**
   - 打开 PuTTY
   - Host Name: `47.xxx.xxx.xxx`（你的公网IP）
   - Port: `22`
   - Connection type: **SSH**
   - 点击 **Open**

3. **登录**
   - login as: `root`
   - password: 输入密码（不会显示，直接输入后按回车）

### Mac 用户

1. **打开终端**
   - 按 `Command + 空格`，输入 `终端`

2. **连接**
   ```bash
   ssh root@47.xxx.xxx.xxx
   ```

3. **输入密码**
   - password: 输入密码（不会显示）

## 步骤3：一键部署（3分钟）

### 方法一：使用一键部署脚本（推荐）

1. **下载部署脚本**
   
   在服务器上执行：
   ```bash
   cd /root
   wget https://raw.githubusercontent.com/你的用户名/仓库名/main/deploy-aliyun.sh
   # 或者手动上传 deploy-aliyun.sh 到服务器
   ```

2. **运行脚本**
   ```bash
   chmod +x deploy-aliyun.sh
   bash deploy-aliyun.sh
   ```

3. **按提示操作**
   - 选择是否使用 Git 克隆项目
   - 如果使用 Git，输入 GitHub 仓库地址
   - 如果不用 Git，手动上传项目文件后再运行脚本

### 方法二：手动部署

如果你不用一键脚本，按照以下步骤手动操作：

```bash
# 1. 更新系统
apt update && apt upgrade -y

# 2. 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt-get install -y nodejs

# 3. 安装 PM2
npm install -g pm2

# 4. 安装 Git
apt install git -y

# 5. 克隆项目（或手动上传）
cd /root
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名

# 6. 安装依赖
npm install

# 7. 创建 .env 文件
cat > .env << EOF
PORT=3000
NODE_ENV=production
JWT_SECRET=$(node -e "console.log(require('crypto').randomBytes(32).toString('hex'))")
JWT_EXPIRES_IN=24h
DB_DIALECT=sqlite
EOF

# 8. 初始化数据库
DB_DIALECT=sqlite node init-db.js

# 9. 启动应用
DB_DIALECT=sqlite pm2 start app.js --name my-website
pm2 save
pm2 startup

# 10. 开放端口
ufw allow 3000/tcp
```

## 步骤4：配置安全组（2分钟）

1. **进入阿里云控制台**
   - 登录阿里云
   - 进入 **云服务器ECS** → **实例**
   - 点击你的服务器实例

2. **配置安全组**
   - 点击 **安全组** 标签
   - 点击安全组ID（如：`sg-xxx`）
   - 点击 **入方向** 标签
   - 点击 **添加安全组规则**

3. **添加规则**
   ```
   端口范围: 3000/3000
   授权对象: 0.0.0.0/0
   描述: 允许访问网站端口
   ```
   - 点击 **保存**

## ❓ 遇到问题？

1. **网站打不开**
   - 检查安全组是否开放 3000 端口
   - 检查防火墙：`ufw status`
   - 检查服务状态：`pm2 status`

2. **数据库错误**
   - 重新初始化：`DB_DIALECT=sqlite node init-db.js`

3. **内存不足**
   - 增加内存限制：`pm2 restart my-website --update-env --node-args="--max-old-space-size=2048"`

**更多问题？** 查看 `阿里云部署详细指南.md`
