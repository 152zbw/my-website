# 服务器启动指南

## 方法1：使用启动脚本（最简单）

1. 打开 **终端（Terminal）** 应用
2. 复制粘贴以下命令并回车：

```bash
cd "/Users/zade/Desktop/JW 2" && ./start-server.sh
```

## 方法2：手动启动

1. 打开 **终端（Terminal）** 应用
2. 复制粘贴以下命令并回车：

```bash
cd "/Users/zade/Desktop/JW 2"
DB_DIALECT=sqlite node app.js
```

## 启动成功的标志

启动成功后，你应该看到类似这样的输出：

```
上传目录: /Users/zade/Desktop/JW 2/uploads
服务器正在运行，端口: 3000
访问地址: http://localhost:3000
API地址: http://localhost:3000/api
数据库模型同步成功
```

## 访问地址

启动成功后，在浏览器中访问：

- **前台网站**: http://localhost:3000
- **后台管理**: http://localhost:3000/html/admin/dashboard-new.html

## 常见问题解决

### 1. 端口被占用

如果看到 `EADDRINUSE` 错误，说明3000端口被占用。

**解决方法**：
```bash
# 查找占用3000端口的进程
lsof -ti:3000

# 杀死占用端口的进程
lsof -ti:3000 | xargs kill -9

# 然后重新启动服务器
```

### 2. 依赖未安装

如果看到 `Cannot find module` 错误，需要安装依赖：

```bash
cd "/Users/zade/Desktop/JW 2"
npm install
```

### 3. 数据库未初始化

如果看到数据库相关错误，运行：

```bash
cd "/Users/zade/Desktop/JW 2"
npm run init-db
```

### 4. 权限问题

如果启动脚本无法执行，运行：

```bash
chmod +x "/Users/zade/Desktop/JW 2/start-server.sh"
```

## 停止服务器

在运行服务器的终端窗口中，按 `Ctrl + C` 停止服务器。
