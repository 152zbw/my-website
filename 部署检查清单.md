# ✅ 部署前检查清单

在开始部署前，请逐项检查以下内容：

## 📋 基础检查

- [ ] **本地测试通过**
  - [ ] 网站能在本地正常运行（`http://localhost:3000`）
  - [ ] 后台能正常登录
  - [ ] 能正常添加/编辑内容
  - [ ] 文件上传功能正常

- [ ] **密码已修改**
  - [ ] 默认密码 `admin123` 已修改为强密码
  - [ ] 新密码已妥善保存

- [ ] **文件完整**
  - [ ] `app.js` 存在
  - [ ] `package.json` 存在
  - [ ] `config/` 目录存在
  - [ ] `html/` 目录存在
  - [ ] `models/` 目录存在
  - [ ] `routes/` 目录存在

## 🔐 安全检查

- [ ] **环境变量配置**
  - [ ] `.env` 文件已创建
  - [ ] `JWT_SECRET` 已设置为强密码（至少32字符）
  - [ ] `NODE_ENV` 设置为 `production`

- [ ] **敏感信息保护**
  - [ ] `.env` 文件已添加到 `.gitignore`（不会上传到Git）
  - [ ] 数据库文件权限设置正确

## 🚀 部署方式选择

选择一种部署方式：

- [ ] **方式一：云服务器**（推荐，完全控制）
  - [ ] 已购买云服务器
  - [ ] 已获取公网IP
  - [ ] 已安装 Node.js（v18+）
  - [ ] 已安装 PM2
  - [ ] 已配置防火墙

- [ ] **方式二：免费托管平台**（最简单）
  - [ ] 已注册 Railway/Render 账号
  - [ ] 已创建 GitHub 仓库
  - [ ] 已上传代码到 GitHub
  - [ ] 已配置环境变量

## 📦 部署步骤

### 如果使用云服务器：

- [ ] **连接服务器**
  - [ ] 已使用 SSH 连接到服务器
  - [ ] 能正常执行命令

- [ ] **安装环境**
  - [ ] Node.js 已安装（`node -v` 显示版本）
  - [ ] npm 已安装（`npm -v` 显示版本）
  - [ ] PM2 已安装（`pm2 -v` 显示版本）

- [ ] **上传代码**
  - [ ] 代码已上传到服务器（Git 或 FTP）
  - [ ] 项目目录结构正确

- [ ] **配置项目**
  - [ ] 已创建 `.env` 文件
  - [ ] 已运行 `npm install`
  - [ ] 已运行 `node init-db.js` 初始化数据库

- [ ] **启动服务**
  - [ ] 已使用 PM2 启动服务
  - [ ] 已设置开机自启
  - [ ] 服务运行正常（`pm2 status` 显示 online）

- [ ] **配置网络**
  - [ ] 防火墙已开放端口（3000）
  - [ ] 域名已解析（如果使用域名）
  - [ ] Nginx 已配置（如果使用 Nginx）

### 如果使用免费托管平台：

- [ ] **准备代码**
  - [ ] 已创建 `.env.example` 文件
  - [ ] 已创建 `Procfile` 文件（Railway 需要）
  - [ ] 代码已上传到 GitHub

- [ ] **配置平台**
  - [ ] 已在平台创建项目
  - [ ] 已连接 GitHub 仓库
  - [ ] 已配置所有环境变量
  - [ ] 已设置启动命令

- [ ] **部署**
  - [ ] 部署已成功
  - [ ] 已获取访问地址

## ✅ 部署后验证

- [ ] **网站访问**
  - [ ] 首页能正常打开
  - [ ] 页面样式正常显示
  - [ ] 图片能正常加载

- [ ] **后台功能**
  - [ ] 后台登录页面能打开
  - [ ] 能正常登录
  - [ ] 能正常添加/编辑内容
  - [ ] 能正常删除内容

- [ ] **API 测试**
  - [ ] API 接口能正常访问
  - [ ] 数据能正常保存
  - [ ] 文件上传功能正常

- [ ] **性能检查**
  - [ ] 页面加载速度正常
  - [ ] 服务器资源使用正常
  - [ ] 没有明显错误日志

## 🔧 故障排查准备

- [ ] **日志查看**
  - [ ] 知道如何查看 PM2 日志（`pm2 logs`）
  - [ ] 知道如何查看服务器日志

- [ ] **备份准备**
  - [ ] 知道如何备份数据库
  - [ ] 已设置定期备份计划

- [ ] **联系支持**
  - [ ] 已保存云服务商客服联系方式
  - [ ] 已保存部署文档链接
