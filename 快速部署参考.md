# ⚡ 快速部署参考（5分钟版）

> 这是简化版，详细步骤请查看 `小白部署指南.md`

## 🎯 最简单的方法：Railway（推荐小白）

### 步骤 1：准备代码（2分钟）

1. **创建 GitHub 账号**（如果没有）
   - 访问 https://github.com
   - 注册账号

2. **上传代码到 GitHub**
   ```bash
   # 在你的项目文件夹里执行
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/仓库名.git
   git push -u origin main
   ```

### 步骤 2：部署到 Railway（3分钟）

1. **访问 Railway**
   - 打开 https://railway.app
   - 用 GitHub 账号登录

2. **创建项目**
   - 点击 **New Project**
   - 选择 **Deploy from GitHub repo**
   - 选择你的仓库

3. **配置环境变量**
   - 点击项目 → **Variables**
   - 添加：
     ```
     PORT=3000
     NODE_ENV=production
     JWT_SECRET=你的随机密钥（至少32字符）
     JWT_EXPIRES_IN=24h
     DB_DIALECT=sqlite
     ```

4. **等待部署**
   - Railway 会自动部署
   - 等待 2-5 分钟

5. **获取地址**
   - 点击 **Settings** → **Generate Domain**
   - 复制地址，如：`https://xxx.up.railway.app`

### 步骤 3：初始化数据库

部署后，数据库需要初始化。你需要：

1. **修改 `package.json`**，添加：
   ```json
   {
     "scripts": {
       "start": "node app.js",
       "postinstall": "DB_DIALECT=sqlite node init-db.js || true"
     }
   }
   ```

2. **重新提交代码**
   ```bash
   git add package.json
   git commit -m "Add database init"
   git push
   ```

3. **Railway 会自动重新部署**

### ✅ 完成！

访问你的地址，应该能看到网站了！

**后台地址：** `https://你的地址/html/admin/index.html`
- 用户名：`admin`
- 密码：`admin123`（**请立即修改！**）

## 🖥️ 云服务器部署（快速版）

### 前提条件
- 已购买云服务器（阿里云/腾讯云）
- 已获取公网IP和密码

### 一键部署脚本

**在服务器上执行：**

```bash
# 1. 更新系统
apt update && apt upgrade -y

# 2. 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt-get install -y nodejs

# 3. 安装 PM2
npm install -g pm2

# 4. 克隆项目（替换为你的 GitHub 地址）
cd /root
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名

# 5. 安装依赖
npm install

# 6. 创建 .env 文件
cat > .env << EOF
PORT=3000
NODE_ENV=production
JWT_SECRET=$(node -e "console.log(require('crypto').randomBytes(32).toString('hex'))")
JWT_EXPIRES_IN=24h
DB_DIALECT=sqlite
EOF

# 7. 初始化数据库
DB_DIALECT=sqlite node init-db.js

# 8. 启动服务
DB_DIALECT=sqlite pm2 start app.js --name my-website

# 9. 设置开机自启
pm2 startup
pm2 save

# 10. 开放端口
ufw allow 3000
ufw enable
```

### ✅ 完成！

访问：`http://你的IP:3000`

## 🔧 常用命令

### PM2 命令
```bash
pm2 status              # 查看状态
pm2 logs my-website     # 查看日志
pm2 restart my-website  # 重启
pm2 stop my-website     # 停止
```

### 数据库备份
```bash
cp config/database.sqlite config/database.sqlite.backup.$(date +%Y%m%d)
```

### 更新代码
```bash
git pull
npm install
pm2 restart my-website
```

## ❓ 遇到问题？

1. **网站打不开**
   - 检查服务是否运行：`pm2 status`
   - 检查端口是否开放：`ufw status`

2. **数据库错误**
   - 重新初始化：`DB_DIALECT=sqlite node init-db.js`

3. **内存不足**
   - 增加内存：`NODE_OPTIONS="--max-old-space-size=2048" pm2 restart my-website`

**更多问题？** 查看 `小白部署指南.md` 的 [常见问题解决](#常见问题解决) 部分。
