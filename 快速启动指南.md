# 快速启动指南

## 第一步：安装 Node.js

您的系统上需要先安装 Node.js 才能运行这个项目。

### Mac 系统安装 Node.js

**方法1：使用 Homebrew（推荐）**
```bash
# 如果还没有安装 Homebrew，先运行：
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 然后安装 Node.js
brew install node
```

**方法2：直接下载安装包**
1. 访问 https://nodejs.org/
2. 下载 LTS 版本（推荐）
3. 双击安装包，按照提示完成安装

**方法3：使用 nvm（Node Version Manager）**
```bash
# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 重新打开终端，然后安装 Node.js
nvm install --lts
nvm use --lts
```

### 验证安装

安装完成后，打开终端（Terminal），运行以下命令验证：

```bash
node --version
npm --version
```

如果显示版本号，说明安装成功！

## 第二步：启动项目

### 方法1：使用启动脚本（最简单）

在终端中，进入项目目录，然后运行：

```bash
cd "/Users/zade/Desktop/JW 2"
chmod +x start.sh
./start.sh
```

### 方法2：手动启动

如果启动脚本无法运行，可以手动执行：

```bash
# 1. 进入项目目录
cd "/Users/zade/Desktop/JW 2"

# 2. 安装依赖（首次运行需要）
npm install

# 3. 初始化数据库（首次运行需要）
node init-db.js

# 4. 启动服务器
npm start
```

## 第三步：访问网站

启动成功后，您会看到类似以下信息：

```
服务器正在运行，端口: 3000
访问地址: http://localhost:3000
API地址: http://localhost:3000/api
```

然后在浏览器中访问：
- **前端网站**: http://localhost:3000
- **后台管理**: http://localhost:3000/html/admin/dashboard.html

## 常见问题

### 1. 提示 "command not found: node"
- 说明 Node.js 未安装或未添加到 PATH
- 请按照上面的步骤安装 Node.js
- 安装后可能需要重新打开终端

### 2. 端口 3000 被占用
- 可以修改 `.env` 文件中的 `PORT=3000` 为其他端口，如 `PORT=3001`
- 或者关闭占用 3000 端口的其他程序

### 3. npm install 很慢或失败
- 可以使用国内镜像：
  ```bash
  npm install --registry=https://registry.npmmirror.com
  ```

### 4. 数据库初始化失败
- 删除 `config/database.sqlite` 文件后重新运行 `node init-db.js`

## 停止服务器

在运行服务器的终端窗口中，按 `Ctrl + C` 停止服务器。

## 需要帮助？

如果遇到问题，请检查：
1. Node.js 版本是否为 14.0 或更高
2. 是否在正确的项目目录中
3. 终端是否有错误信息
